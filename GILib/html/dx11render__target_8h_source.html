This file contains the interfaces used to define render target resources.
gi_lib::IRenderTargetArray
Base interface for render target arrays. A render target array is an array of textures, each of which can be drawn upon. Elements in the array are guaranteed to have the same dimensions. The array also defines an optional shared depth stencil buffer for Z testing while drawing.
