Detailed Description

template<typename TElement>
class gi_lib::StructuredArray< TElement >

Represents a low-level buffer that behaves like a strongly-typed array of elements. This array can be written by a CPU and read by a GPU.

Template Parameters
TElementType of the elements stored within the array.
