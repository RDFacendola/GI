Detailed Description

template<typename TIterator>
class gi_lib::Range< TIterator >

Wraps a pair of iterators defining a range that can be iterated through.

Template Parameters
TIteratorType of the iterators to wrap. Any forward iterator compilant with the standard template library should work.
