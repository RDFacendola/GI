Detailed Description

Represents a single point light. A point light is a light that has a position and irradiates light in all directions. The position of the light is defined by a separate transform component. The attenuation of the point lights is defined as Att(d) = (Kc + Kl*d + Kq*d*d)^-1 where Kc is a constant factor Kl is a linear factor Kq is a quadratic factor d is the distance of the surface from the light.

Raffaele D. Facendola.
