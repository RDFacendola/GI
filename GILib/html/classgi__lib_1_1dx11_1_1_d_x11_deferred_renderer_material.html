Detailed Description

Material for a DirectX11 deferred renderer. A custom material should not be compiled from code directly since there's no way of knowing whether the code is compatible with the custom renderer. A concrete deferred material is not a subclass of a DirectX11 material to prevent a DDoD. Composition is preferred here.
