Detailed Description

Base interface for general-purpose textures. A general-purpose resource can be accessed by the GPU for both reading and writing purposes. A general purpose texture is a decorator over a standard Texture2D. It is NOT a derivation of a texture.

Author
Raffaele D. Facendola.
