Detailed Description

template<typename TDirect, typename TIndex, typename TType>
struct gi_lib::IndexedIterator< TDirect, TIndex, TType >

Wraps a collection and an iterator. The iterator is used to visit the collection in a non-linear way.

Template Parameters
TDirectType of the collection. Must support the bracket operator: TDirect::operator[](...) -> TType&
TIndexType of the index iterator. Must be compliant with an input iterator.
TTypeType of elements referenced by the collection. See TDirect.
